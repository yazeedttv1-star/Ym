# 🚀 مشروع يزيد أحمد ابوالحمد المتكامل (Profile & Database)

هذا المستودع يحتوي على الموقع التفاعلي الخاص بـ **يزيد أحمد ابوالحمد**، مدمج مع نظام تسجيل الدخول الحقيقي وقاعدة البيانات السحابية باستخدام **Firebase**.

## 🛠️ التقنيات المستخدمة
* **HTML5 / CSS3**: لتصميم الواجهات والـ Slides بأسلوب عصري (Dark Mode).
* **JavaScript (ES6+)**: للتحكم في منطق اللعبة والتنقل بين الشاشات.
* **Firebase Auth**: لتسجيل الدخول الآمن بحسابات Google.
* **Firebase Realtime Database**: لحفظ تقدم المستخدمين، كلمات السر، ونسب التفاعل لايف.

## 📦 هيكل قاعدة البيانات (Database Schema)
